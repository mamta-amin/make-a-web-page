World Wide Web

World Wide Web is collection of web pages which are made up of text files. These files contain image, text, links to other pages, video, and HTML tags. A browser uses these text documents along with other files for styling and scripting to display what we see on the screen.

HTML Basics

HTML (Hypertext Markup Language) is a language used by a browser to display a web page. HTML is made up of series of tags that hold instructions for the computer. Tags are basically commands for the browser. HTML document can only contain text and can be created using editors like Sublime, Notepad++, etc. As you write any code it is important to remember not to repeat code since all that code has to be maintained. Don't Repeat Yourself (DRY) is one of the fundamental principal of programming.
Each web page has a basic structure that consists of some base tags like HTML, head, and body. Other sub tags branch out from these base tags forming a tree like structure. Most tags come in pair of an opening and closing tags; however there are some that have no closing tags. Below are some examples of tags and their function. Cascading Style Sheet

CSS (Cascading Style Sheet) is a language used to create presentation of a webpages. While HTML allows you to structure your content CSS allows you to keep your content separate from your styling. Style can be included in a web page by linking the CSS file in the HTML document or by writing style in-line with HTML. Last option is not preferred since it may cause a lot of repeated code. When you create CSS file, certain styling can be applied to same class that may appear several times in your HTML file and thus reducing repetition.
